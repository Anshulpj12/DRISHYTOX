How It Works

Seven steps. Zero network required after the first.

1

🟢 Online — GPS Tracking

GPS coordinates active, crossing history built silently in background. Under 120 bytes per crossing.

2

🟡 Signal Drops — Dead Reckoning

Historical dead reckoning activates. Uses personal crossings, community data, OBD-II speed, or phone sensors — in that priority.

3

⏸ Stop Detected

Accelerometer or OBD-II detects vehicle stop. Block counter pauses. Stop duration excluded from distance calculation.

4

🚨 SOS Triggered

Ultra-compressed packet built: block code + type + time + confidence — under 50 bytes. Sent via SMS, USSD, or LoRa.

5

📡 V2V Relay

Passing vehicles carry AES-256 encrypted packets via BLE/WiFi Direct. No driver interaction. Auto-forwarded on network restore.

6

🖥 Server Decodes

Packet decoded, provider matched by category + coverage zone + availability. Best provider selected automatically.

7

✅ Provider Dispatched

Provider receives alert, accepts dispatch, navigates to block. User gets ETA confirmation. Event tracked until resolution.
