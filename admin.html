🔄 Config Push System

Real-time provider update distribution to driver apps via lightweight version polling

How It Works

• When you add, remove, or toggle a provider, the config version is automatically bumped.

• Driver apps poll every 30 minutes, checking a 4-byte version number (near-zero bandwidth).

• If the version has changed, the driver app downloads only the diff (added/removed providers).

• Result: zero manual intervention — provider changes propagate automatically.
